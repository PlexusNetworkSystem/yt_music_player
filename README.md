# youtube music player
This is a backgorund youtube music player.
# :arrow_double_down: Installation :arrow_double_down:
```
wget https://github.com/PlexusNetworkSystem/yt_music_player/archive/refs/heads/main.zip
unzip main.zip
sudo rm -f main.zip
echo done
```
